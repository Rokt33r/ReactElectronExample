# React Electron example app

일렉트론 상에서의 최신 기술 사용 예제입니다.

## 사용 스택

- **ReactHotLoader3**
- **Webpack2**
- **Styled Components**
- Electron
- React
- React Router
- Redux

굵게 표시된 것이 최근 정식으로 나왔거나 곧 나올 스택들 입니다.
